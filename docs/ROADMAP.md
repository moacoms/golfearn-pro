## 2. 레슨프로 찾기 개선 (2026-04-06 기획)

현재 `/find-pro` 페이지는 프로 전체 나열만 하고 검색/필터가 없음. 서비스화 전 필수.

### Step 1. DB 컬럼 추가 (Supabase SQL Editor)
```sql
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS pro_grade text;
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS pro_lesson_venues text[];
```

프로 등급 값:
| DB 값 | 표시명 |
|---|---|
| `semi_pro` | 세미프로 |
| `lesson_pro` | 레슨프로 |
| `kpga_associate` | KPGA 준회원 |
| `kpga_member` | KPGA 정회원 |
| `tour_pro` | 투어프로 |

### Step 2. 프로 프로필 편집 UI
추가 필드:
- 프로 등급 (`pro_grade`) — 드롭다운
- 레슨 장소 (`pro_lesson_venues`) — 복수 입력 칩 (예: "고양CC 인도어", "청담동 TGR")
- 레슨 지역 (`pro_location`) — 자유 입력 (예: "서울 강남")
- 전문분야 (`pro_specialties`) — 복수 선택 칩 (스윙교정, 숏게임, 퍼팅, 드라이버, 입문자, 주니어)
- 레슨비 (`pro_monthly_fee`) — 숫자 입력

### Step 3. 검색 + 필터
파일: `lib/features/dashboard/presentation/pages/find_pro_page.dart`
- 검색바 (이름, 지역, 소개글 통합)
- 지역 필터 칩 (DB에서 `pro_location` 고유값 추출)
- 경력 필터: 전체 / 1~3년 / 3~5년 / 5~10년 / 10년 이상
- 정렬: 이름순(기본), 경력순
- 참고 패턴: `students_list_page.dart` 검색/필터 칩

### Step 4. 프로 카드 & 상세
- 카드: 등급 배지, 레슨 장소, 요약
- 상세 다이얼로그: 전체 소개, 전문분야 칩, 자격증, 레슨비, 연락처, 레슨 신청 버튼
