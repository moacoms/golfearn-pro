# 레슨프로 찾기 기능 개선 기획

## 배경
현재 "레슨프로 찾기" 페이지는 프로 전체 나열만 하고 검색/필터가 없음.
프로 등급, 레슨 장소 등 핵심 정보도 부족하여 학생이 원하는 프로를 찾기 어려움.

---

## 현재 상태

### 페이지
- **파일**: `lib/features/dashboard/presentation/pages/find_pro_page.dart`
- 전체 프로 목록 나열 (이름순), 검색/필터/상세 없음

### DB 필드 (profiles 테이블)

| 필드 | 용도 | 활용 | 입력 UI |
|------|------|------|---------|
| `full_name` | 이름 | O | O |
| `pro_phone` | 전화번호 | O | O |
| `pro_introduction` | 소개글 | O (2줄) | O |
| `pro_experience_years` | 경력(년) | O | O |
| `pro_location` | 지역 | O | **X** |
| `pro_specialties` | 전문분야 | X | **X** |
| `pro_certification` | 자격증 | X | **X** |
| `pro_monthly_fee` | 레슨비 | X | **X** |
| **(신규)** `pro_grade` | 프로 등급 | - | **신규** |
| **(신규)** `pro_lesson_venues` | 레슨 장소 | - | **신규** |

---

## 작업 순서

### Step 1: DB 컬럼 추가 (Supabase SQL Editor)

```sql
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS pro_grade text;
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS pro_lesson_venues text[];
```

**프로 등급 값:**

| DB 값 | 표시명 |
|--------|--------|
| `semi_pro` | 세미프로 |
| `lesson_pro` | 레슨프로 |
| `kpga_associate` | KPGA 준회원 |
| `kpga_member` | KPGA 정회원 |
| `tour_pro` | 투어프로 |

### Step 2: 프로 프로필 편집 UI 추가

프로 설정 페이지에 추가할 필드:
- **프로 등급** (`pro_grade`) — 단일 선택 드롭다운
  - 세미프로 / 레슨프로 / KPGA 준회원 / KPGA 정회원 / 투어프로
- **레슨 장소** (`pro_lesson_venues`) — 복수 입력 칩
  - 예: "고양CC 인도어", "청담동 TGR", "일산 골프존"
- **레슨 지역** (`pro_location`) — 자유 입력
  - 예: "서울 강남", "경기 고양"
- **전문분야** (`pro_specialties`) — 복수 선택 칩
  - 스윙교정 / 숏게임 / 퍼팅 / 드라이버 / 입문자 전문 / 주니어 전문
- **레슨비** (`pro_monthly_fee`) — 숫자 입력

### Step 3: 텍스트 검색 + 지역 필터

- 검색바: 이름, 지역, 소개글, 레슨 장소 통합 검색
- 지역 필터 칩: DB에서 `pro_location` 고유값 동적 추출
- 참고 패턴: `students_list_page.dart`의 검색바 + 필터 칩

### Step 4: 경력 필터 + 정렬

- 경력 범위: 전체 / 1~3년 / 3~5년 / 5~10년 / 10년 이상
- 정렬: 이름순(기본), 경력순

### Step 5: 프로 카드 개선

- 등급 배지 표시 (예: [KPGA 정회원])
- 레슨 장소 표시 (예: 고양CC 인도어, 청담동 TGR)

### Step 6: 프로 상세 다이얼로그

카드 클릭 시 표시:
- 이름 + 등급 배지
- 경력, 지역
- 레슨 장소 목록
- 전체 소개글
- 전문분야 칩, 자격증, 레슨비, 연락처
- 레슨 신청 버튼
