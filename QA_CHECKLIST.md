# QA 체크리스트 (2026-04-01)

배포 완료된 기능들의 확인 체크리스트입니다.

---

## 1. 학생 관리 페이지 (`/students`)

- [ ] 학생 목록이 이름 가나다순으로 정렬되는지
- [ ] 상단에 "레슨 중 / 레슨 종료 / 전체" 필터 칩 3개 표시
- [ ] "레슨 종료" 선택 시 삭제된 학생만 반투명으로 표시
- [ ] 그룹 지정한 학생이 있으면 그룹 필터 칩 행 표시
- [ ] 학생 카드에 보라색 그룹 배지 표시

## 2. 학생 등록/수정 (`/students/new`, 학생 편집)

- [ ] "그룹" 입력 필드 표시 (예: A반, 오전반, VIP)
- [ ] "가족 연결" 드롭다운으로 다른 학생 선택 가능
- [ ] 가족 연결 후 저장 정상 동작

## 3. 학생 상세 (`/students/:id`)

- [ ] 하단에 "결제 내역" 섹션 표시
- [ ] 총 결제 금액 초록색 박스로 표시
- [ ] 개별 결제 내역 (날짜, 결제수단, 금액) 표시
- [ ] 가족 연결했으면 "그룹/가족 정보" 섹션 표시
- [ ] 가족 구성원 탭하면 해당 학생 상세로 이동

## 4. 스케줄 페이지 (`/schedule`)

### 반복 레슨
- [ ] 레슨 추가 시 "반복 레슨" 스위치 ON
- [ ] 반복 주기 드롭다운: 매일 / 매주 / 격주 / 매월 4개 옵션
- [ ] 매월 선택 시 같은 날짜로 다음 달 생성되는지

### 레슨 완료 자동화
- [ ] 레슨 카드 탭 → "레슨 완료" 선택
- [ ] 완료 후 수입 관리(`/income`)에 자동 기록 확인
- [ ] 완료 후 학생 상세에서 레슨 횟수 +1 확인
- [ ] 완료 후 "다음 레슨 예약" 제안 다이얼로그 표시
- [ ] 다이얼로그에서 "다음 레슨 예약" 버튼 동작 확인

### 예정 복원
- [ ] 완료된 레슨 → "예정으로 복원" 선택
- [ ] 복원 시 학생 레슨 횟수 -1 되는지

## 5. 수입 관리 (`/income`)

- [ ] 수입 기록이 날짜별로 그룹핑 표시
- [ ] 각 날짜 헤더 오른쪽에 일별 소계 금액 표시
- [ ] 상단 월간 총 수입 정상 표시

## 6. 레슨 노트 (노트 탭 → + 버튼)

- [ ] 학생 선택 전: AI 버튼 회색 비활성
- [ ] 학생 선택 후: "AI 레슨 노트 자동 생성" 초록 그라데이션 버튼 활성
- [ ] AI 버튼 터치 → 로딩 표시 → 전체 필드 자동 채움
  - [ ] 레슨 내용
  - [ ] 핵심 포인트
  - [ ] 개선 사항
  - [ ] 과제/숙제
  - [ ] 다음 레슨 포커스
- [ ] 자동 생성 후 수정 가능한지
- [ ] 수정 후 저장 정상 동작

## 7. 대시보드 (`/home`)

- [ ] 상단 딥 그린 → 에메랄드 그라데이션 헤더
- [ ] 헤더에 프로 이름 + 골프 아이콘
- [ ] 패키지 만료 알림: 골드 강조 디자인 (해당 패키지 있을 때)
- [ ] 카드들에 부드러운 그림자 적용

## 8. 전체 디자인

- [ ] 하단 네비게이션: 플로팅 스타일 (둥근 모서리, 그림자)
- [ ] 선택된 탭에 초록 배경 + 볼드 텍스트
- [ ] 앱 전체 색상이 딥 그린(#065F46) 기반으로 통일
- [ ] 버튼, 스낵바, 아이콘 등 일관된 색상
