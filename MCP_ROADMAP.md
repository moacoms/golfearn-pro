# MCP & 자동화 로드맵

Golfearn 앱에 적용할 MCP 연동 및 자동화 기능 구현 계획입니다.

---

## Phase 1: 앱 내 자동화 (MCP 불필요, 즉시 구현 가능)

### 1-1. 레슨 완료 → 자동 수입 기록
- **현재**: 레슨 완료 처리 후 수입을 별도로 수동 입력
- **개선**: 레슨 완료 시 패키지 금액 기반으로 자동 수입 기록 생성
- **구현**: `schedule_page.dart` 완료 액션에서 income 자동 생성 로직 추가
- **파일**: `schedule_page.dart`, `income_repository_impl.dart`

### 1-2. 패키지 만료 알림 강화
- **현재**: 대시보드에서 만료 임박 패키지 표시
- **개선**: 앱 실행 시 팝업 알림, 잔여 1회 시 강조 표시
- **구현**: 대시보드 진입 시 패키지 체크 → 다이얼로그 표시
- **파일**: `dashboard_page.dart`

### 1-3. 학생 진도 자동 추적
- **현재**: total_lesson_count 수동 관리
- **개선**: 레슨 완료 시 자동 카운트 증가 + last_lesson_at 업데이트
- **구현**: 레슨 완료 핸들러에서 student 업데이트 추가
- **파일**: `schedule_page.dart`, `student_repository_impl.dart`

### 1-4. 다음 레슨 자동 제안
- **현재**: 매번 수동으로 다음 레슨 예약
- **개선**: 반복 레슨 패턴 분석 → "다음 레슨 예약하시겠습니까?" 제안
- **구현**: 레슨 완료 후 다이얼로그로 다음 레슨 시간 제안
- **파일**: `schedule_page.dart`

## Phase 2: Google Calendar 연동 (MCP)

### 2-1. 레슨 스케줄 → Google Calendar 동기화
- **목적**: 프로가 기존 구글 캘린더에서도 레슨 일정 확인
- **방향**: 단방향 (앱 → 구글) 먼저, 이후 양방향
- **MCP**: Google Calendar API
- **구현 순서**:
  1. Google OAuth 연동 (설정 페이지에 "캘린더 연결" 버튼)
  2. 레슨 생성/수정/삭제 시 캘린더 이벤트 자동 생성/수정/삭제
  3. 캘린더 이벤트에 학생 이름, 레슨 타입, 장소 포함
- **파일**: 새 서비스 `google_calendar_service.dart`, `schedule_page.dart`

### 2-2. Google Calendar → 앱 가져오기 (Phase 2.5)
- 구글 캘린더의 변경사항을 앱에 반영
- 충돌 처리 로직 필요

---

## Phase 3: Gmail 연동 (MCP)

### 3-1. 월간 리포트 이메일 발송
- **목적**: 매월 학생에게 레슨 요약 리포트 자동 발송
- **내용**: 이번 달 레슨 횟수, 진도, 프로 코멘트
- **MCP**: Gmail API
- **구현 순서**:
  1. Gmail OAuth 연동
  2. 월간 리포트 HTML 템플릿 생성
  3. 설정에서 "월간 리포트 발송" 토글
  4. 매월 1일 자동 발송 (또는 수동 발송 버튼)
- **파일**: 새 서비스 `gmail_service.dart`, `settings_page.dart`

### 3-2. 레슨 영수증 발송
- 수입 기록 시 학생에게 영수증 이메일 자동 발송
- 선택적 기능 (설정에서 ON/OFF)

---

## Phase 4: 카카오톡 알림톡 연동

### 4-1. 레슨 리마인더
- **목적**: 레슨 1시간 전 학생에게 카카오 알림톡 발송
- **내용**: "OO님, 오늘 14:00에 레슨이 예정되어 있습니다"
- **구현**: 카카오 알림톡 API 연동 (비즈니스 채널 필요)
- **구현 순서**:
  1. 카카오 비즈니스 채널 개설
  2. 알림톡 템플릿 등록 (검수 필요)
  3. Supabase Edge Function으로 스케줄러 구현
  4. 레슨 1시간 전 자동 발송
- **파일**: Supabase Edge Function, 새 서비스

### 4-2. 패키지 만료 알림
- 잔여 2회 이하 시 학생에게 알림톡 발송
- "레슨 패키지가 곧 소진됩니다. 갱신을 원하시면 프로에게 문의해주세요"

### 4-3. 레슨 완료 알림
- 레슨 완료 시 학생에게 요약 발송
- "오늘 레슨이 완료되었습니다. 숙제: [프로 입력 내용]"

---

## Phase 5: Google Sheets 연동 (MCP)

### 5-1. 수입 데이터 자동 백업
- **목적**: 세금 신고용 수입 데이터를 스프레드시트로 자동 관리
- **MCP**: Google Sheets API
- **구현 순서**:
  1. Google Sheets OAuth 연동
  2. 수입 기록 생성 시 자동으로 시트에 행 추가
  3. 월별 시트 자동 생성
  4. 설정에서 연결된 시트 URL 표시
- **파일**: 새 서비스 `google_sheets_service.dart`

### 5-2. 학생 데이터 백업
- 학생 목록을 별도 시트에 동기화
- 현재 CSV 내보내기의 자동화 버전

---

## Phase 6: AI 고도화 (Claude API)

### 6-1. 레슨 노트 AI 초안 생성
- **현재**: 프로가 수동으로 레슨 노트 작성
- **개선**: 학생 정보 + 이전 노트 기반으로 AI가 초안 생성
- **구현**: 레슨 완료 시 "AI 노트 생성" 버튼 → Claude API 호출
- **프롬프트**: 학생 레벨, 목표, 이전 노트, 레슨 타입 기반 맞춤 초안

### 6-2. 학생 맞춤 연습 추천
- 레슨 노트 이력 분석 → 맞춤 연습 프로그램 자동 생성
- 학생 대시보드에 "이번 주 연습 추천" 카드 표시

### 6-3. 스윙 분석 고도화
- 현재 이미지 분석 → 영상 분석 확장
- 분석 이력 저장 및 변화 추적

---

## Phase 7: Supabase Realtime

### 7-1. 실시간 알림
- 프로가 레슨 완료 → 학생 앱에 즉시 반영
- 학생이 프로 찾기에서 레슨 신청 → 프로에게 실시간 알림
- **구현**: Supabase Realtime 채널 구독

### 7-2. 실시간 스케줄 동기화
- 프로가 스케줄 변경 시 학생 앱에 즉시 반영
- 변경 시 인앱 토스트 알림

## "스마트 레슨 플로우" 최종 목표

```
레슨 시간 1시간 전
  → 카카오 알림톡으로 리마인더 (Phase 4)
  
레슨 완료 버튼 터치
  → 패키지 횟수 자동 차감 (기존)
  → 수입 자동 기록 (Phase 1-1)
  → 학생 레슨 카운트 자동 증가 (Phase 1-3)
  → Google Calendar 이벤트 완료 표시 (Phase 2)
  → AI 레슨 노트 초안 생성 (Phase 6-1)
  → 학생에게 레슨 요약 카카오 발송 (Phase 4-3)
  → 다음 레슨 자동 제안 (Phase 1-4)
  → Google Sheets에 수입 자동 기록 (Phase 5)
```
